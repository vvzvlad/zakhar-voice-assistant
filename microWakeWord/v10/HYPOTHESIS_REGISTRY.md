# Реестр гипотез/идей «захааар» (v1–v9) — проверено / отброшено / не проверено

Сводка из ВСЕХ отчётов всех версий (v1/v4/v5/v7/v8/v9 + RESEARCH 2–5 + G-заметки агента).
Цель — видеть, что уже сделано и **что можно делать СЕЙЧАС, пока пишутся device-семплы**.

## Статус
Продакшен — **v8**. На едином leakage-safe device-eval (153 held-out, cutoff 0.9):
device-FRR **0.059** (recall 0.941), FAPH с VAD = 0. История на одной шкале (G9):
v1 FRR 0.346 → v4 0.163 → v5 0.183 (мелкий регресс) → **v8 0.059**.
Худшие режимы (стратифицированно): `music_snr5` 0.216, `reverb+music10` 0.176, `muffled` 0.157.

| **STT-фильтр синтетики** (Vosk) | убрал 62% мусора (сломанный TTS) |

## ❌ ОТБРОШЕНО (и почему)
| идея | почему отброшено |
|------|------------------|
| negative_class_weight 30 (v2) | пере-консервативна, recall упал |
| adversarial mining раунды 2–3 | плато |
| **Knowledge distillation (logit)** | без выигрыша на нашем масштабе (нужно >400k позитивов) |
| **Focal loss (γ=2)** | регресс |
| **margin-BCE (logit-shift аппрокс)** | регресс — НО это НЕ настоящий angular head (см. «не проверено») |
| **Residual connections** | ВРЕДЯТ (0.065→0.137 на тех же данных) |
| **SLR28 RIR-банк** | слишком гулкий, вредит (mit_rirs лучше) |
| **Агрессивная multi-condition аугментация** | пере-портит (плато ~20×) |
| **Синтетический music-mix в обучение** | НЕВЕРНОЕ распределение: устройство гонит AEC по loopback → KWS видит ОСТАТОК эха, не полную музыку. Регресс |
| Тяжёлые SOTA (Keyword Mamba 3.4M, KWT 5M) | не влезают в 32k/45КБ |
| Синтетика тишины/шума как негатив тишины-бага | у обеих моделей 0.0 ложных — баг от РЕАЛЬНОЙ микроструктуры комнаты, синтетикой не воспроизводится |
