# Аудит загрязнения синтетических позитивов (Vosk ru-small STT) + чистый набор v8

## Баг
Piper на ПЕРЕудлинённых написаниях с повторяющимися гласными («захаааар»,
«захаааааар») ломает фонемизатор и произносит ДРУГИЕ слова (дорога, врага, драка,
два года, товар, трава, сбер…). Голоса Piper **denis/dmitri** ломаются особенно сильно
ДАЖЕ на правильном «Захар». Silero и Piper/irina — чистые.

## Калибровка STT (важно для честности)
Прогон тех же Vosk-проверок по 621 РЕАЛЬНОЙ человеческой записи: **17% помечено
«мусором»**. Это люди → 17% = ШУМ STT (ru-small плохо декодит протяжную речь), НЕ
реальный мусор. Значит истинный мусор синтетики ≈ (помечено − ~17%). Vosk ЗАНИЖАЕТ
(мягкий фильтр оставляет 'за ха' и пр.); ручная проверка оператора давала выше (62% на
v5/drawn_raw).

| набор (версии) | помечено мусором | ≈истинный (−17% floor) | худшие голоса |

Вывод: ВСЕ версии (v1,v4,v5) учились на заметно загрязнённой синтетике; главный
источник — Piper/denis,dmitri на «Захар». irina/ruslan/silero — норм.

## Починка генерации
- gen_v8_piper.py / gen_v8_silero.py: ТОЛЬКО правильные написания «Захар»/«Заха́р»
  (ударение). Протяжность — высокий length_scale (1.3–2.1, корректная медленная речь) +
  опц. тайм-стретч чистого «Захар» (sox tempo, питч сохранён). Никаких повторных гласных.

## Чистый STT-верифицированный набор v8
Сгенерил 15500 правильным написанием, STT-фильтр (Vosk, мягкий) →
**kept 10261 / 15500 (66%)**. Dropped по голосам:
- Piper/dmitri: 97% выброшено (1932/2000) — голос непригоден для «Захар»
- Piper/denis: 78% выброшено
- Piper/ruslan: 53% выброшено
- Piper/irina: 11% выброшено (хороший)
- Silero (все): 6% выброшено (чистый)
Итог чистого набора: 10261 (доминируют Silero 7065 + irina 1773 + ruslan 933).
Список: v8/drawn_v8_clean_list.txt ; клипы: v8/pos_clean/.
